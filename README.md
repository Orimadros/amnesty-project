## Setup

The R environment is initialized with renv. Restore the environment with:

```r
renv::restore()
```

## Project Structure

- `code/` - Data wrangling and analysis scripts
- `data/` - Raw, temporary, and processed data
- `output/` - Results and figures
- `docs/` - Literature, TeX files, and notes
- `r-src/` - Source code modules
